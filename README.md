# CS174A Final Project
Tower Deadfence is a tower defense game inspiried by Bloon Tower TD and Plants vs. Zombies, where you can place towers and try to defend your fence from being attacked from zombies.

# How to Run Locally
1. Extract zip file contents

2. Open project folder in IDE of your choice, and a terminal

3. Change Directory into specfied folder: 

```bash
cd project-folder-name
```

4. Run the following commands: 

```bash
npm install
npm run dev
```

5. Use Cnrl - C or copy the link given, which should look like this:

```bash
 ➜  Local:   http://localhost:0123/
```

6. Project should be running on your preferred browser

# Controls and Instructions for Use
During the building phase (day-time):
- b - Activate building mode
- mousedown - Place tower while in building mode

During zombie mode (night-time):
- f - fastforward game
- s - pause game
- c - switch camera mode
    - Use arrows keys to move around setting
